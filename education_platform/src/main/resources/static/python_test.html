О курсе

Курс "Python для начинающих" предназначен для тех, кто только начинает свой путь в программировании. Вы освоите фундаментальные концепции программирования и научитесь применять их на практике с помощью языка Python.

Содержание курса

В ходе курса вы изучите следующие темы:

  • Модуль 1: Введение в Python и основы синтаксиса
  • Модуль 2: Переменные, типы данных и операции
  • Модуль 3: Условные операторы и циклы
  • Модуль 4: Функции и модули
  • Модуль 5: Работа с файлами
  • Модуль 6: Обработка исключений
  • Модуль 7: Основы ООП в Python
  • Модуль 8: Работа с библиотеками
